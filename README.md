# 🏥 SipadiApp - Sistem Pakar Diagnosa Hipertensi

Aplikasi berbasis web untuk mendiagnosa tingkat penyakit hipertensi menggunakan metode *Dempster-Shafer*.

### 🧠 Tentang Project
Project ini mengimplementasikan teori probabilitas *Dempster-Shafer* (Theory of Evidence) untuk menangani ketidakpastian dalam diagnosa medis. Sistem akan menghitung bobot dari setiap gejala yang dialami untuk menentukan hasil diagnosa yang paling akurat.

### 🛠️ Tech Stack
- *Language:* PHP
- *Database:* MySQL
- *Metode:* Dempster-Shafer Logic

### 🚀 Fitur
- Input gejala klinis pasien secara interaktif.
- Perhitungan nilai densitas (m) otomatis.
- Hasil diagnosa tingkat risiko hipertensi
